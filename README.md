Steps to updating the mod

---

-1: If you don't want to do this then ask Kerkel to do it

0: Make sure you're on the latest pull

1: Copy this mod's folder to your desktop

2: Remove the following files from the now copied folder:

    1: The hidden .git folder

    2: .gitignore

    3: README.md

    4: .vscode folder and .luarc.json if present

3: Select the copied folder's metadata.xml from the mod uploader, add change notes, and upload mod

4 (IMPORTANT): copy the metadata.xml from the copied folder into the original folder, and push the updated metadata.xml. If you don't, the next update to this mod may not be pushed out to users
